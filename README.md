# Retail Sales Analysis and Prediction

## Project Overview

This project analyzes a real-world retail sales dataset to identify business trends and predict sales using Machine Learning. The analysis includes data preprocessing, exploratory data analysis (EDA), visualization, and predictive modeling to gain valuable business insights.

---

## Objectives

* Analyze retail sales data to identify trends and patterns.
* Explore relationships between sales, profit, quantity, and discount.
* Visualize sales performance across different categories and regions.
* Build a Machine Learning model to predict sales.
* Evaluate model performance and identify important features.

---

## Dataset Information

The dataset contains retail order details, including:

* Order Date
* Category
* Sub-Category
* Product Name
* Sales
* Quantity
* Discount
* Profit
* Region
* Customer Information

## Key Insights

* Technology products generated the highest sales.
* Profit varied significantly across product categories.
* Sales differed across regions, highlighting regional demand.
* Discounts influenced overall profit.
* Quantity and discount were useful features for predicting sales.
