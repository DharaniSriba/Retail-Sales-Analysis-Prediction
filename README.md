# Retail-Sales-Analysis-Prediction
# Images Folder

This folder contains all visualizations generated during the **Retail Sales Analysis and Prediction** project.

## Visualizations Included

- Sales by Category
- Profit by Category
- Sales by Region
- Monthly Sales Trend
- Correlation Heatmap
- Top 10 Products by Sales
- Feature Importance
- Prediction Output

These visualizations were created using **Python**, **Matplotlib**, and **Seaborn** to analyze retail sales data, identify business trends, evaluate the machine learning model, and present meaningful insights.
